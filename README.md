# Famosos Dluz

Portal de celebridades sem WordPress, feito em Next.js e PostgreSQL.

## Publicar

Depois de publicar no Coolify, abra `/admin`. Defina as variáveis `DATABASE_URL` e `ADMIN_PASSWORD` no serviço. O painel cria a tabela de notícias automaticamente na primeira publicação.

## Coolify

Crie um serviço PostgreSQL e uma aplicação a partir deste repositório. Na aplicação, informe `DATABASE_URL` com a conexão interna do PostgreSQL e `ADMIN_PASSWORD` com uma senha forte. Use a porta `3000` e associe o domínio `famosos.dluz.com.br`.
